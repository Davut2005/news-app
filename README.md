# 📰 NewsPulse 

A lightweight, responsive news aggregator built with **React** and **TypeScript**. This application fetches real-time headlines from a public News API and displays them in a clean, modern interface.

## 🚀 Features
* **Real-time Data:** Fetches the latest stories using the [NewsAPI](https://newsapi.org/) (or similar).
* **Type Safety:** Fully typed props and API responses using TypeScript.
* **Responsive Design:** Optimized for mobile, tablet, and desktop views.
* **Search & Filter:** Quickly find articles by keywords or categories.

## 🛠️ Tech Stack
* **Framework:** React 18
* **Language:** TypeScript
* **Styling:** Tailwind CSS / Styled Components
* **Data Fetching:** Axios / TanStack Query
* **Icons:** Lucide React / React Icons
